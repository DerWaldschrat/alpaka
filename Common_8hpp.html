#define ALPAKA_NO_HOST_ACC_WARNING   _Pragma("hd_warning_disable")

Disable nvcc warning: 'calling a host function from host device function.'.

Usage: ALPAKA_NO_HOST_ACC_WARNING device host function_declaration()

It is not possible to disable the warning for a host function if there are calls of virtual functions inside. For this case use a wrapper function. WARNING: only use this method if there is no other way to create runnable code. Most cases can solved by #ifdef CUDA_ARCH or #ifdef CUDACC.
