Calls the given function object fnObj count times trying to utilize vectorization with TVectorizationCount elements.

The supported number of elements per vector of a given type on the current architecture can be queried with registerGetVectorizationSizeElems<TElem>::value. For optimal vectorization, the fnObj should be only consist of one line.
