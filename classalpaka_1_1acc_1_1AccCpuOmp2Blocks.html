Detailed Description

template<typename TDim, typename TSize>
class alpaka::acc::AccCpuOmp2Blocks< TDim, TSize >

The CPU OpenMP 2.0 block accelerator.

This accelerator allows parallel kernel execution on a CPU device. It uses OpenMP 2.0 to implement the grid block parallelism. The block size is restricted to 1x1x1.
