Detailed Description

template<typename TDim, typename TSize>
class alpaka::acc::AccCpuOmp4< TDim, TSize >

The CPU OpenMP 4.0 accelerator.

This accelerator allows parallel kernel execution on a CPU device. It uses CPU OpenMP4 to implement the parallelism.
