Detailed Description

template<typename TDim, typename TSize>
class alpaka::acc::AccCpuThreads< TDim, TSize >

The CPU threads accelerator.

This accelerator allows parallel kernel execution on a CPU device. It uses C++11 std::thread to implement the parallelism.
