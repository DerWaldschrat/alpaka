Detailed Description

template<typename TDim, typename TSize>
class alpaka::acc::AccCpuSerial< TDim, TSize >

The CPU serial accelerator.

This accelerator allows serial kernel execution on a CPU device. The block size is restricted to 1x1x1 and all blocks are executed serially so there is no parallelism at all.
